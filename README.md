# week1-arenadevelopers
This project is a responsive personal portfolio website created using HTML5. It highlights fundamental web development concepts such as semantic HTML, navigation, forms, accessibility, and clean code organization.
Personal Portfolio Website
Description

A personal portfolio website built using HTML5 as part of Week 1 internship training.
This project demonstrates basic web development concepts such as HTML structure, semantic elements, internal navigation, image handling, and form creation.

Project Objectives

Learn the fundamentals of HTML

Understand webpage structure and markup language

Use semantic HTML elements correctly

Create internal navigation links

Build a basic contact form

Technologies Used

HTML5

Visual Studio Code

Google Chrome Browser

Project Structure
portfolio-website/
│
├── index.html
├── README.md
├── images/
│   └── profile.jpg
├── requirements.txt (optional)

HTML Concepts Implemented

HTML document structure (<!DOCTYPE>, <html>, <head>, <body>)

Headings, paragraphs, links, and images

HTML attributes (id, class, href, src, alt)

Semantic HTML tags (header, nav, main, section, article, footer)

Forms with input validation

Website Sections

Header: Displays name and navigation menu

About: Personal introduction with profile image

Skills: List of technical skills

Projects: Showcases featured projects

Contact: Contact form with name, email, and message fields

Footer: Copyright information

Setup Instructions

Download or clone the repository

Open the project folder in Visual Studio Code

Open index.html in any web browser

Ensure images are placed inside the images folder

Testing and Validation

Tested in Google Chrome

Verified internal navigation links

Checked form input validation

Ensured images load correctly

Reviewed semantic HTML structure

Learning Outcomes

Gained understanding of HTML5 fundamentals

Learned how to structure a webpage using semantic tags

Improved knowledge of forms and navigation

Understood basic accessibility practices

Conclusion

This project provided hands-on experience in building a structured and semantic HTML webpage.
It serves as a strong foundation for learning CSS and JavaScript in future internship weeks.
